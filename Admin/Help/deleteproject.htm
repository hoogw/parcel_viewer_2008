Delete Record
Previous  Top  Next



The Delete Record utility enables administrators to quickly and easily delete obsolete or incorrect projects from the GeoPrise.NET Project Coordination application.

ðClick the Projects Tab and select the Delete Record option to open the Delete Project Search dialog  
 
 
 
Project data can be queried based on the following one or more of the following criteria:  

·Project ID  
·Project Name      
·Project Owner      
·Project Division      
·Project Type      
·Project Start and End Date      
·Contact Name      
·Project Description      
·Contract Number  
 
ðClick the 'Find Projects' button to display a tabular result of all projects matching the above search parameters  
 
 
 
ðClick the green hyperlink of the project to be edited to display the Project Data Delete dialog confirmation  
 
GeoPrise.NET will zoom to the project extents in the main map display for visual verification of the project to be deleted  
 
 
 
ðClick the 'Delete This Project' button to complete the task
