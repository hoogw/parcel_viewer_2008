The Application Tables dialog affords application administrators the ability to effectively update and manage all GeoPrise.NET Project Coordination data lookup-up tables.

ðFrom the Account Administration window, select the Application Tables hyperlink to display the Look-Up Table Administration dialog  
 
 
 
ðSelect the desired hyperlink to update the corresponding data look-up table  
 
Tables available for editing include:  
·   Project Owner Table - Owners types for all projects can be added, deleted, edited, and/or viewed. Examples of project owners are: Department of Transportation, County, etc.  
     
·   Project Status Table - Status options for projects can be added, deleted, edited, and/or viewed. Examples of project status are: Construction Project, Water Quality Project, etc.  
     
·   Project Contact Types Table - Contact types can be added, deleted, edited, and/or viewed. Examples of contact types are: Contractor, County Inspector, Consultant, etc.  
     
·   Project Divisions Table - Various divisions within the 'Project Owner' table can be added, deleted, edited, and/or viewed. Examples of divisions can be: Transportation Division, Water Quality Division, Construction Division, etc.  
 
·   Project Type Table - Project types can be added, deleted, edited, and/or viewed. Examples of project types are: Water Supply Project, Construction Permit Project, Transportation Improvement Project, etc.
