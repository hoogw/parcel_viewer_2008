Edit Record
Previous  Top  Next



The Edit Record utility enables administrators to quickly and easily edit existing project data (both spatial and tabular) within the GeoPrise.NET Project Coordination application.

ðClick the Projects Tab and select the Edit Record option to open the Edit Project Search dialog  
     
 

Project data can be queried based on the following one or more of the following criteria:  

·Project ID  
·Project Name      
·Project Owner      
·Project Division      
·Project Type      
·Project Start and End Date      
·Contact Name      
·Project Description      
·Contract Number  
 
ðClick the 'Find Projects' button to display a tabular result of all projects matching the above search parameters  
     
 
 
ðClick the green hyperlink of the project to be edited to display the Enter New Project Data dialog  
 
 

GeoPrise.NET will zoom to the project extents in the main map display to enable the user to not only effect tabular changes but to add and remove associated project street segments as well.  
 
ðWhen all project data has been updated, click the 'Submit' button to commit the changes
