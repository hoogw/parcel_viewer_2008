Project Contacts

The Project Contacts tool provides administrators with instant access for editing, deleting, adding, and viewing current project point of contact information.
 
ðFrom the Account Administration window, select the Project Contacts hyperlink  
 
ðSelect the desired hyperlink to update the Project Contact data  
 
 
 
 
 
1.   Add a Project Contact - Project contacts can be added by simply filling in the appropriate information and clicking the 'Add Record' button     
 
 
 
2.   Edit a Project Contact - Project contacts can be edited by updating the following information and clicking the 'Edit Record' button  
 
 
 
 
3.   Delete a Project Contact - Project contacts can be deleted by querying for the appropriate contact and then clicking the 'Delete Record' button  
 
 
 
 
4.   Re-Assign a Project Contact - Project contacts are re-assigned by entering contact information into the following dialog and clicking the 'Re-assign' button  
 
 
 
5.   View all Records - Select this option to further review all projects
