Coordinate Projects
Previous  Top  Next



The Coordinate Projects tool allows project administrators to select projects based on pre-determined criteria, establish whether or not those projects overlap any other scheduled projects at the same location, check for any coordination conflicts and, if needed, establish links to facilitate project synchronization.

ðFrom the Account Administration window, select the Coordinate Projects hyperlink to open the Project Coordination dialog  
 
 
 
ðInput the information relating to the project(s) to be coordinated and click the 'Find Projects' button to display a tabular result of all projects matching the above search parameters  
 
 
 
ðClick the green hyperlink of the project to be coordinated to display the Coordination dialog  
 
 
 
The above dialog returns a tabular listing of projects occuring within the same time frame and at the same location as the project just queried and also indicates whether or not those projects are currently being coordinated.  
 
Coordinated projects can now be linked with the selected project by checking the 'Coordinated?' box and adding any additional user 'Comments'
