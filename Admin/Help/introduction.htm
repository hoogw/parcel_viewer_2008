Welcome to the GeoPrise.NET Administrator Help Site!

The GeoPrise.NET administrator functionality is designed to operate as a GIS Portal or Enterprise Gateway to access, create, and edit organizational information both spatially and through various data input formats. GIS data, and data from other business information systems, can be integrated, presented, edited, and accessed through the GeoPrise.NET Administrator application. In this application, project specific information can be queried, added, and edited to present detailed spatial project data via the GeoPrise.Net viewer. Once logged in as an administrator, this functionality is available via the Projects Tab.
