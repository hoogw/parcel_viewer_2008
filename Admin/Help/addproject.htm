The Add Record utility enables administrators to quickly and easily add new projects to the GeoPrise.NET Project Coordination application.

ðClick the Projects Tab and select the Add Record option to open the Add New Project dialog  
 
 
 
ðClick the 'Go To Step 1' button to begin the project creation process  
 
 
 
ðUse the easy to follow steps presented in the Select Streets dialog to add street segments associated with the new project  
 
ðClick the 'Go To Step 2' button to access the Enter New Project Data dialog  
 
 
 
The Enter New Project Data dialog facilitates entry of specific project data including:  

·Project Name      
·Project Owner      
·Project Division      
·Project Type      
·Project Start and End Date      
·Contact Name      
·Project Description      
·Contract Number      

Note: All fields are required. Once the appropriate project information has been entered into the dialog, select the 'Submit' button to commit the data.
