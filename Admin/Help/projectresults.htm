Project Results

The Project Results tool enables project administrators to toggle on/off and order columns displayed whenever project attribute values are displayed within the GeoPrise.NET Project Coordination application.

ðFrom the Account Administration window, select the Project Results hyperlink to open the new window  
 
ðCheck/uncheck the 'Display?' box(es) relating to 'Column Names' to toggle on/off whenever project details are displayed  
 
ðArrange the project detail attribute columns by re-numbering the 'Order #' box(es) accordingly (column ordering occurs from left to right ascending numerically).
