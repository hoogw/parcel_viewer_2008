Measure Distance
Top  Previous  Next



The Measure Distance tool enables users to identify the distance between two or more defined points on the main map display.

ðUse the navigational tools to locate a specific area of interest  
 
ðClick the Select Tab and choose the Measure Distance tool to open the new window  
 
ðUse the mouse to select the origin of the first line segment  
 
ðOnce the point is displayed on the map, select a destination point to end the first line segment  
The distance of each segment will be displayed in the pop-up window as the 'Current' distance  
 
ðMore segments can be added to the original line by selecting additional map points with the mouse pointer  
The cumulative distance of all segments will be displayed in the pop-up window as the 'Total' distance
