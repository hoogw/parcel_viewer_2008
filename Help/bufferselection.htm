The Buffer tool enables users to specify a selected set of data by defining a line, polygon, or rectangle entity, creating a zone of a specified distance around those features (the buffer), and then selecting all corresponding data within that proximity.
 
ðUse the navigational tools to locate a specific area of interest  
 
ðClick the Select Tab and choose the Buffer tool to display the 'Buffer Selection' dialog  
 
 
 
ðActivate the radio button of the corresponding selection option to be used for the buffer definition  
 
·   Select by Line  
ðCreate a line of data by selecting two or more points on the main map display with the mouse pointer (a red dot will be created with each click to depict the user specified points. GeoPrise.NET will also create adjoining lines between these entities as the linear selection is shaped  
 
ðWhen finished, click the 'Complete & Select' button  
GeoPrise.NET will then select and display all data intersecting the newly created buffer line  
 
 
 
·   Select by Polygon  
 
ðCreate a buffer polygon by clicking and selecting multiple data points with the mouse. GeoPrise.NET will automatically create the polygon defined by these user specified points and will select and display all data encompassed within the new area  
 
ðWhen finished, click the 'Complete & Select' button  
 
 
 
·   Select by Rectangle  
 
ðCreate a buffer rectangle by clicking and dragging a box to select the appropriate location. GeoPrise.NET will automatically create the rectangle and select all data within the new area  
 
ðWhen finished, click the 'Complete & Select' button  
 
 
 
Once the range of data has been selected and highlighted, a new dialog box will request the buffer distance and buffer layer.  
 
 
 
Upon selecting the appropriate distance and layer, GeoPrise.NET will present the attribute data for all matching records.
