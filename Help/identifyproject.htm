The Identify Project tool allows users to select those projects which have already been identified using:

1.    The Search Project search option  
2.    Navigational tools to locate a specific area and then selecting Display Options from the Overview Tab.  
 
ðClick the Select Tab and choose the Identify Project tool  
 
ðSelect the radio button in the new Identify Project window indicating the data you would like to retrieve  
 
ðA ? will appear next to the mouse cursor once the Identify Project tool is selected. Simply click on a colored project in the viewer to open a new window with additional project details  
 
 

ðSelect the 'Project Name' green hyperlink to display specific project details or select the 'Contact' hyperlink to display current contact/project owner information
