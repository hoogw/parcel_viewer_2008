The Search Address option enables users to query for specific parcels by entering a valid street address or range of addresses.

ðClick the Search Tab and select the Search Address option to display the query dialog  
 
ðEnter a valid house number and street name into the dialog box  
Refer to the Address Search Tips for more detailed help.  
 
ðClick the 'Submit' button  
GeoPrise.NET will now begin searching for parcels matching the above criteria. Results are displayed in tabular format within a new window  
 
ðSelect from the candidate parcels by clicking on the parcel link. GeoPrise.Net will now zoom to the selected parcel within the map interface.  

In addition, depending upon your organizational data sets and version of GeoPrise.NET, user queries may be based upon the following criteria:

·Vacant parcels - The application will present all vacant parcels at a provided street address  
·Address Ranges - The application will present all parcels within a provided address range  
·One or more street name characters - For use in instances where proper spelling of a street name is unsure  

ðSelect 'Click Here' within the search address dialog box to change to one of the different query options mentioned above.  

Address Search Example:

1) Enter a known address or range of addresses and click 'Find Addresses'



2) A new window will appear presenting the detailed parcel information in a tabular format and the GeoPrise.NET application will zoom to the specified parcel in the main map display. Additional detailed parcel information can be accessed by selecting any of the green data links within the Parcel Details window.



3) To search by 'Vacant Parcels', simply select that option from the main 'Search Address' window and a new window will appear. Enter the vacant parcel street name and a Parcel Details window will present the specific information.



4) To search by 'Address Range', simply select that option from the main 'Search Address' window and a new window will appear. Enter the starting address, end address, street name, and select 'Find Addresses'.



GeoPrise.NET will present a listing of the parcels in the selected range (see below). Click on an individual Parcel ID to retrieve Parcel Details
