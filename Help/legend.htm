The Legend window outlines information currently being presented within the GeoPrise.NET application and depicts the symbology used to display each Layer.

ðClick the Overview Tab and select the Legend option to open the new window  
 
 

·   Information presented within the Legend is dependent upon the currently selected Layers and will change dynamically whenever the map Layers'' visibility is modified and refreshed.  
 
·   Legend information is also scale dependent. Consequently, only Layers currently visible within the main map will be displayed in the Legend.  

Note: Adding and removing Layers within the Legend is dependant upon the GeoPrise.NET installation.
