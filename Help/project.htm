The Search Project search option enables users to select application projects based upon Project Name, Project Owner, Project Division, Project Type, and active Date Ranges.


ðClick the Search Tab and select the Search Project option to display the query dialog  
 
 
 
 
ðSelect the desired project criteria utilizing the dialog drop down selections and/or by selecting a start/end date from the interactive calendar.  
 
ðSelect 'Find Projects'  
 
GeoPrise.NET will present a new Project Search Results window listing all projects meeting the above specifications  
 
 
 
ðSelect any individual hyperlink to display additional details and zoom to the selected project within the map display  
 
 
 
ðSelect the 'Contact Info' hyperlink to display current contact/project owner information
