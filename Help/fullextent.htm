The Full Extent tool returns the map display to the maximum extents of all application data layers.

ðClick the Navigate Tab and select the Full Extent option to activate the tool  


Note: When zoomed to the Full Extent of the map display, not all Layers may draw due to scale dependencies. In order to view all Layers, select one of the navigational/search tools to view the data at a more detailed level.
