Identify Parcel
Top  Previous  Next



The Identify Parcel tool enables a user to select a parcel from the map in order to display APN, address, and generalized owner information.

ðUse the navigational tools to locate a specific area of interest  
 
ðClick the Select Tab and choose the Identify Parcel tool  
 
ðA ? will appear next to the mouse cursor once the Identify Parcel tool is selected. Simply click on any parcel to open a new window with the general parcel attribute information  
 
 
 
To select more detailed attribute information, refer to the Identify Parcel Details tool.  

 Tip: This functionality can also be accessed by selecting the tool from the Standard Toolbar.
