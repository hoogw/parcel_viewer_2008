The Search APN option enables users to query for specific parcels by the APN (Assessor Parcel Number) or via a range of APNs.

ðClick the Search Tab and select the Search APN option to display the query dialog  
 
ðEnter a valid parcel number into the 'From" column of the dialog box in the provided xxx-xxxx-xxx-xxxx format. To search a range of APNs, enter a second APN in the 'To' column  
Refer to the APN Search Tips for more detailed help.  
 
     
 
ðClick the 'Submit' button  
GeoPrise.NET will now begin searching for parcels matching the above criteria. Results are displayed in tabular format within a new window.  
 
ðSelect from the candidate parcels by clicking on the parcel link  
GeoPrise.Net will now zoom to the selected parcel within the map interface.
