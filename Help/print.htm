The Print option easily transitions GeoPrise.NET into 'Print Mode'

ðClick the Report Tab and choose the Print option to open the new window and expose the following options:  
 
·   Page Size (default page sizes are available in addition to a custom page size definition option)      
·   Map Scale (scale in inches to feet at which the map will be displayed)      
·   Orientation (landscape vs portrait)      
·   Color Mode (color vs black/white)  
 
     
 
While in 'Print Mode', GeoPrise.NET offers the utilities noted below.  
 
ðSimply move the cursor over any of the following toolbar selections to access their functionality:  
 
Zoom In, Zoom Out, Pan, Refresh, Preview, Print, Save, Change  
 
           
Note: In order to enable the Print, Save or Change options, you must first select the Preview button  
 
Tip: This functionality can also be accessed by selecting the tool from the Standard Toolbar.
