Map Layers represent spatial data available for viewing within the GeoPrise.NET application. The check boxes next to each layer indicate whether or not that layer is currently being drawn within the main map display.

ðClick the Overview Tab and select the Layers option to open the new window  
 
 

To change the information being displayed within the map, simply check the box next to a layer to enable it, or uncheck its box to disable it, and click the 'Refresh' button. GeoPrise.NET will then automatically rebuild the map display to reflect the new configuration.

Tip: This functionality can also be accessed by selecting the tool from the Standard Toolbar.
